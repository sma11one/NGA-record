### 0 64792920 2023-03-06 19:51
今天晚上下班回家，说是有我的信。

其实听到家人说“收到信”的时候，我就能猜到发件人是我的她，也就是我的第一任，果不其然，打开信是请帖，是她亲笔写的，我不知道我是不是特殊的一个，但是我还是很高兴。

她是我的第一个，也是陪伴我时间最长，留给我遗憾最多的女朋友，改变我第二多的女性。(第一当然是我老妈，嘿嘿)有人问我一个男生，是怎么懂各种各样护肤，健身，化妆等等很多男生不接触的东西，答案全在她身上。

我们相遇是在大学入学，我记得我入学时应该是秋天，在排队等学长学姐分新生大巴的时候，就看到一个女生坐在行李箱上，穿着一双带着马丁靴在那里一晃一晃，身上穿一件小风衣，头发压的散直，当时我还是个高考刚刚结束，出产自男女比例悬殊理科班的害羞男生，就在那里假装环顾地偶尔扫一眼。

不过没有我最爱的日本轻小说里的四目相对，或者古诗里所说的心有灵犀一点通，她完全没看我就上车了，我也觉得也就如此别过，幸运的话可能会在学校食堂里相遇，能偶尔看上一眼，毕竟在一个工科学校，boy meet girl的剧情，不属于平平无奇的我。

结果就很巧的，我在电子五，她在电子四。

当然也很不巧的，我们班20:1，她们班20:2。

对了，她是团委，她们班还有一个185以上，长得又白又帅的班长。

后来合班上大课，大家一开始互相不熟悉，也就自己寝室坐在一起，男生一堆一堆，女生一群一群，也就是那时，我才发现，原来“不要脸”的力量原来这么大。

她那时还穿着牛仔裤和小皮靴。其实当时我很紧张。我觉得如果像我这么普普通通的人坐在他旁边，有什么不好的行为她可能会反感，甚至在脑中想着，如果真的让她反感了，我看她一眼都是罪过，所以上课时很谨小慎微。

也就在第一节课下课的时候，我掏出了手机开始打一个“不知死活”的游戏，就在我的屏幕上出现了一个脖子上带着一个法式圆圈马桶套脖饰的小loli的时候，一个带着一点点口音的“粗犷”(其实也不是很粗，就是不像我想象中的那样甜)嗓音传来

我能听出是个女生，还以为是哪个大姐头，结果转头一看……她那个和声音完全不符的形象就在歪头看着我。

实话实说，吓了我一大跳。

当时我的内心疯狂吐槽：颜值诈骗。

我记得她接视频电话之后我开口的第一句话就是“向伟大的劳动女性工作者致以崇高的敬意，祝你妇女节快乐！”(其实就是整活)。

然后她二话不说就在一脸黑线中把视频挂了，哈哈哈哈哈

但当时我也算是经历了恋爱之后成长很多，也没有急的给她回过去，只是慢悠悠地收拾桌子，但是手机还是铃声和振动一起开着

果然就在几分钟之后，她把视频打了回来，然后我就在一种微微的紧张中收拾收拾衣服，用湿巾擦了擦手，接了她的视频。

开屏之后她直接和我打趣：“这么多年都没给我过次女生节，上来就过妇女节，真有你的哈”

我反唇相讥：“你早就不是女生了，想过女生节建议穿越”(其实这个玩笑如果和不熟悉的女生开有点猥琐)

不过她倒是完全没脸红，只是一脸鄙夷地“也不知道哪个家伙暑假不回家，还要人陪噢。”

我回她“也不知道是哪个笨蛋听到另一个笨蛋说不回家之后偷偷租了一个季度的房子”

然后我俩就你一句“笨蛋”我一句“蠢材”地插科打诨。

后来我给他直播我收拾办公室，她和我说“没想到你这么爱干净”，我嘲讽她“也不知道浴室的头发是收拾的”

但是很默契的，我和她都没有说一些过线的话，因为她知道我单身，而我也知道她有男朋友。

也就在这样一个没什么人陪伴的下午，一个新入职的学生和一个小米手机中传来的声音，度过了人生中最有意义的一天之一。


其实往后每年我都会在妇女节前一天给她祝福，祝她“明天妇女节快乐”，她每次都调侃我“未来人你好”/“礼物呢？”/“我还很年轻！”，但每次我基本都哈哈一下然后聊点别的，因为我知道，她的妇女节不需要我祝福。
