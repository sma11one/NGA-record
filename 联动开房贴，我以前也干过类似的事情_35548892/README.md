## 联动开房贴，我以前也干过类似的事情

### 0 64831775 2023-03-03 13:14
高中的时候，疯狂追求一个女生。她身材特别好，家境也很好，基本上是我们学校第一批穿AJ和supreme这些潮牌的。我们学校当时离市区特别远，放学后其他同学都有家里接送，但是她家直接在学校旁边的别墅区买了别墅，所以她还是我们学校当时为数不多的可以走路回家的同学。

当时我和她一个班，就是前后桌。然后经常互相讨论题目，互相教解题这些，关系越发热络起来。当时我觉得我自己真行了，就打算攒钱给她买礼物，然后等她生日冲塔。但是那个时候我特别穷，因为父母发现我有时候会拿钱买点卡，就把我零花钱断了。所以我就在学校里给人当牛做马疯狂从富哥们手里挣钱。

有一次，她看到我放学以后去帮别人值日，就站在门口问我为什么帮这个人值日。我当时想都没想就说想攒钱给喜欢的人买东西。她听完鬼魅一笑，笑着对我说那也别把自己累坏了。我听完以后感觉如沐春风，感觉自己简直是全世界最幸福的人。

差不多过了两周吧，就在我钱马上攒够的时候，我又借了一个周五放学后帮忙打扫图书室的活。我打扫完后已经是晚上快九点了。因为我是骑车回家，我担心要是拖的太晚不安全。我就打算从穿过学校篮球馆直接抄近道回家。(因为如果正常走的话，从图书室到校门得绕个大圈)然后当我路过篮球馆更衣室的时候，我听到了类似于嗯嗯啊啊的声音。其实一开始听到的时候我以为是闹鬼，还是有点害怕的。但是我压低脚步走进门口的时候，我发现这个动静好像是两个人在做。我压低脚步，把耳朵凑到离门十几厘米的距离，想听听是谁在做。

然后发生的事情就很经典了，我发现那个声音就是她的声音。和她做的那个人应该是某个有更衣室钥匙的特长生。我大概能猜到是谁，但是这个时候纠结这些已经没意义了。我把耳朵从门边移开，当我转头准备走的时候，我看到更衣室门外的鞋柜上摆着她那双AJ，款式和配色我记得很清楚，就是2017年发售的AJ1情人节配色，鞋后面有一个心形的篮球刺绣。

我攒着给她买礼物的钱，我考上大学以后都充在LOL的皮肤抽奖里面了。大四某天刷得物的时候，好死不死地又在首页刷到了那双鞋，我点开页面，发现这双鞋42码的卖2599。我当时也不知道是气急败坏还是贼心不死吧，居然鬼迷心窍地下了单。鞋很快送到了我手上，可我并不想穿，直到今天，装着它的纸盒仍然静静地躺在我家的储藏室里。
